# Українське радіо (online) 📻

Простий та зручний додаток для прослуховування українських радіостанцій. Програма написана мовою Python з використанням сучасної бібліотеки **PyQt6**, що забезпечує стабільну роботу графічного інтерфейсу та аудіоплеєра без необхідності встановлювати сторонні програми (наприклад, VLC).

## Основні функції

- **Швидкий вибір радіостанцій**: Популярні українські хвилі (Українське Радіо, Промінь, Культура, Хіт FM, Люкс ФМ, Радіо Максимум) вже додані до випадаючого списку.
- **Вибір якості потоку**: Для окремих станцій (наприклад, Суспільного) можна перемикати між "Високою" та "Низькою" якістю, що ідеально для повільного інтернету.
- **Вбудований аудіоплеєр**: Використовує вбудовані у Windows мультимедійні кодеки (через PyQt6 `QMediaPlayer`), відтворюючи аудіо прямо з коробки. 
- **Будильник / Авто-розклад**: Налаштуйте автоматичне увімкнення або вимкнення радіо за розкладом!
  - Оберіть конкретні дні тижня.
  - Вкажіть діапазон часу (наприклад: з `08:00` до `18:00`), протягом якого радіо має грати. Поза цим часом воно буде автоматично зупинено.
- **Згортання в системний трей**: При закритті (натисканні на "Х") вікно не закривається, а акуратно ховається біля годинника. З іконки в треї можна відновити вікно, поставити плеєр на паузу, або повністю закрити програму.
- **Автозапуск з Windows**: Можливість додати програму в автозавантаження (через налаштування в самій програмі). Радіо запускатиметься автоматично після увімкнення комп'ютера.
- **Дві теми оформлення**: Світла та Темна тема (зберігається в `radio_config.json`).

## Встановлення та Запуск

1. Переконайтеся, що на вашому комп'ютері встановлено **Python (версія 3.10 або вище)**.
2. Запустіть файл `_START_.bat`.
   - Цей скрипт автоматично перевірить наявність Python.
   - Створить віртуальне середовище `.venv`.
   - Встановить необхідну бібліотеку `PyQt6`.
   - Запустить додаток.

При наступних запусках `_START_.bat` програма буде відкриватися миттєво, оскільки все налаштування зберігатиметься локально.

## Технічні вимоги

- ОС: Windows 10/11 (вбудовані медіа кодеки).
- Інтернет-з'єднання для відтворення радіостанцій.
- Бібліотека `PyQt6` (встановлюється автоматично). Не потрібен ні VLC, ні додаткові плеєри!
