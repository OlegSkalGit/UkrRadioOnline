# Українське радіо (online) 📻

Простий та зручний додаток для прослуховування українських радіостанцій. Програма написана мовою Python з використанням сучасної бібліотеки **PyQt6**, що забезпечує стабільну роботу графічного інтерфейсу та аудіоплеєра без необхідності встановлювати сторонні програми (наприклад, VLC).

## Детальний опис функцій та налаштувань

Усі параметри роботи програми можна налаштувати у меню **Налаштування** на верхній панелі:

* **Тема оформлення (Світла / Темна тема)**:
  Швидке перемикання кольорової гами інтерфейсу. Доступна сучасна темна тема (для зменшення навантаження на очі ввечері) та класична світла тема. Налаштування зберігається автоматично.

* **Автопрогравання**:
  При увімкненні цієї опції програма автоматично починає відтворювати останню обрану радіостанцію одразу після запуску додатка (якщо активний розклад дозволяє відтворення в цей час).

* **Автоперемикання джерела при обриві**:
  У разі тимчасової втрати інтернет-з'єднання або проблем на сервері мовлення, плеєр почекає 5 секунд і спробує автоматично перемкнутися на резервне джерело (якщо воно передбачене для цієї станції) або повторити спробу підключення до поточного стріму.

* **Автозапуск з Windows**:
  Додає додаток до списку автозавантаження операційної системи через реєстр Windows. Радіо запускатиметься автоматично при кожному ввімкненні комп'ютера.

* **Автозгортання в трей при запуску**:
  Якщо ця опція активна разом із автозапуском, програма відкриватиметься у згорнутому стані (одразу в системний трей біля годинника), не відволікаючи користувача візуально.

* **Згортати в трей при закритті/згортанні**:
  Корисна функція для фонового прослуховування. При натисканні на кнопку згортання або хрестик закриття вікна, інтерфейс програми хоється в системний трей, але відтворення не переривається. Повністю вийти з програми можна кнопкою **Вихід** у головному меню або через контекстне меню трею.

* **Вибір звукової карти**:
  Дозволяє переспрямувати звук радіо на будь-який доступний аудіопристрій (наприклад, окремо на навушники чи USB-колонки), навіть якщо в системі за замовчуванням встановлено інший пристрій.

* **Увімкнути розклад (автозапуск/зупинка)**:
  Активує режим будильника/автоматичної роботи:
  - **Вибір днів тижня** та **діапазону часу** роботи (наприклад, з `08:00` до `18:00`).
  - **Підтримка нічних змін (перехід через північ)**: можна налаштувати період на кшталт з `22:00` до `06:00` ранку наступного дня.
  - **Захист від випадкового ввімкнення**: при активованій опції спроба вручну ввімкнути радіо поза вказаним часом блокується із попередженням, щоб запобігти раптовому шуму в неробочий час.

## Інші інтегровані можливості

* **Швидкий вибір станцій**: Усі популярні українські хвилі (Українське Радіо, Промінь, Культура, Хіт FM, Люкс ФМ, Радіо ROKS, KISS FM та інші) вже вбудовані у програму.
* **Динамічне оновлення плейлистів**: Програма автоматично у фоновому режимі перевіряє та завантажує свіжі плейлисти. Першочергово завантажуються офіційні стріми з сервера `https://radio.ukr.radio/` (включаючи регіональні станції та якість потоків), а потім плейлисти IPTV від `iptv.org.ua` та збірки `Opencartbot`. Це автоматично розширює перелік доступних хвиль до сотень варіантів.
* **Захист від запуску копій (Single Instance)**: Програма блокує запуск кількох екземплярів одночасно. При спробі відкрити додаток вдруге, перше вікно автоматично розгорнеться на екрані, а в треї з'явиться відповідне сповіщення.

## Встановлення та Запуск

1. Переконайтеся, що на вашому комп'ютері встановлено **Python (версія 3.10 або вище)**.
2. Запустіть файл `_START_.bat`.
   - Цей скрипт автоматично перевірить наявність Python.
   - Створить віртуальне середовище `.venv`.
   - Встановить необхідну бібліотеку `PyQt6`.
   - Запустить додаток.

При наступних запусках `_START_.bat` програма буде відкриватися миттєво, оскільки все налаштування зберігатиметься локально.

## Технічні вимоги

- ОС: Windows 10/11 (вбудовані медіа кодеки).
- Інтернет-з'єднання для відтворення радіостанцій.
- Бібліотека `PyQt6` (встановлюється автоматично). Не потрібен ні VLC, ні додаткові плеєри!
