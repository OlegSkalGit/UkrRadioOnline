# Українське радіо (online) 📻

Простий та зручний додаток для прослуховування українських радіостанцій. Програма написана мовою Python з використанням сучасної бібліотеки **PyQt6**, що забезпечує стабільну роботу графічного інтерфейсу та аудіоплеєра без необхідності встановлювати сторонні програми (наприклад, VLC).

## Основні функції та можливості

### 1. Прослуховування радіостанцій
* **Швидкий вибір станцій**: Усі популярні українські хвилі (Українське Радіо, Промінь, Культура, Хіт FM, Люкс ФМ, Радіо ROKS, KISS FM та інші) вже вбудовані у програму.
* **Перемикання якості (джерела)**: Для окремих станцій (наприклад, Суспільного) можна обрати між "Високою якістю" (192kbps) та "Резервним" потоком меншої якості, що дуже допомагає при слабкому або мобільному інтернеті.
* **Динамічне оновлення плейлистів**: Програма автоматично у фоновому режимі перевіряє та завантажує свіжі плейлисти (IPTV від iptv.org.ua та збірки Opencartbot), додаючи сотні нових регіональних та інтернет-радіостанцій.

### 2. Інтелектуальний розклад (Будильник)
* **Автоматична робота**: Налаштуйте автоматичний запуск або зупинку програвання у конкретні години та обрані дні тижня.
* **Підтримка нічного розкладу**: Алгоритм повністю підтримує перехід через північ (наприклад, з `22:00` вечора до `06:00` ранку), коректно відстежуючи день старту розкладу.
* **Контроль ручного запуску**: Якщо розклад активовано, спроба увімкнути радіо вручну поза встановленим часом буде автоматично заблокована із виведенням інформаційного вікна. Це допомагає запобігти небажаному шуму.

### 3. Автоматичне відновлення зв'язку та стабільність
* **Автоперемикання при збоях**: У разі обриву інтернет-з'єднання або зависання потоку програма почекає 5 секунд і автоматично перемкнеться на альтернативне джерело (якщо воно є) або спробує знову завантажити поточний стрім.
* **Вибір аудіопристрою**: У меню налаштувань можна вибрати конкретну звукову карту (колонки, навушники чи HDMI-вихід) для виведення звуку.

### 4. Зручний інтерфейс та інтеграція з ОС
* **Робота в системному треї (Tray)**: При закритті (натисканні на хрестик) вікно не закривається, а ховається у системний трей біля годинника. З іконки в треї можна керувати плеєром (запуск, пауза, вихід) або відкрити головне вікно.
* **Захист від запуску копій (Single Instance)**: Програма блокує запуск кількох екземплярів одночасно. При спробі відкрити додаток вдруге, перше вікно автоматично розгорнеться на екрані, а в треї з'явиться відповідне сповіщення.
* **Автозапуск з Windows**: Можливість увімкнути автоматичний запуск програми разом зі стартом комп'ютера.
* **Дві теми оформлення**: Швидке перемикання між Темною (Curated Catppuccin-like) та Світлою темами інтерфейсу.
* **Збереження стану**: Усі налаштування (тема, остання станція, рівень гучності, обраний пристрій виводу) зберігаються автоматично у файл `radio_config.json`.

## Встановлення та Запуск

1. Переконайтеся, що на вашому комп'ютері встановлено **Python (версія 3.10 або вище)**.
2. Запустіть файл `_START_.bat`.
   - Цей скрипт автоматично перевірить наявність Python.
   - Створить віртуальне середовище `.venv`.
   - Встановить необхідну бібліотеку `PyQt6`.
   - Запустить додаток.

При наступних запусках `_START_.bat` програма буде відкриватися миттєво, оскільки все налаштування зберігатиметься локально.

## Технічні вимоги

- ОС: Windows 10/11 (вбудовані медіа кодеки).
- Інтернет-з'єднання для відтворення радіостанцій.
- Бібліотека `PyQt6` (встановлюється автоматично). Не потрібен ні VLC, ні додаткові плеєри!
