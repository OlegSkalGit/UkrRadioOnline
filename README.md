# Українське радіо (online) 📻

Простий та зручний додаток для прослуховування українських радіостанцій. Програма написана мовою Python з використанням сучасної бібліотеки **PyQt6**, що## Детальний опис функцій та налаштувань

Усі параметри роботи програми можна налаштувати у вікні **Налаштування**, яке відкривається однойменною кнопкою на верхній панелі. Налаштування згруповані за чотирма вкладками:

### 1. Вкладка "Програвання"
* **Автопрогравання**:
  При увімкненні цієї опції програма автоматично починає відтворювати останню обрану радіостанцію одразу після запуску додатка (якщо активний розклад дозволяє відтворення в цей час).
* **Автоперемикання джерела при обриві**:
  У разі тимчасової втрати інтернет-з'єднання або проблем на сервері мовлення, плеєр почекає 5 секунд і спробує автоматично перемкнутися на резервне джерело (якщо воно передбачене для цієї станції) або повторити спробу підключення до поточного стріму.
* **Звукова карта**:
  Дозволяє переспрямувати звук радіо на будь-який доступний аудіопристрій (наприклад, окремо на навушники чи USB-колонки), навіть якщо в системі за замовчуванням встановлено інший пристрій.
* **Увімкнути розклад (автозапуск/зупинка)**:
  Активує режим будильника/автоматичної роботи. При натисканні на кнопку **Налаштувати розклад** відкривається вікно, де можна налаштувати:
  - Вибір днів тижня та діапазону часу роботи (наприклад, з `08:00` до `18:00`).
  - Підтримку нічних змін (перехід через північ, наприклад, з `22:00` до `06:00` ранку наступного дня).
  - Захист від випадкового ввімкнення: при активованій опції спроба вручну ввімкнути радіо поза вказаним часом блокується із попередженням, щоб запобігти раптовому шуму в неробочий час.

### 2. Вкладка "Запуск та Вікно"
* **Автозапуск з Windows**:
  Додає додаток до списку автозавантаження операційної системи через реєстр Windows. Радіо запускатиметься автоматично при кожному ввімкненні комп'ютера.
* **Автозгортання при запуску**:
  Якщо ця опція активна разом із автозапуском, програма відкриватиметься у згорнутому стані (одразу в системний трей біля годинника), не відволікаючи користувача візуально.
* **Згортати в трей при закритті/згортанні**:
  Корисна функція для фонового прослуховування. При натисканні на кнопку згортання або хрестик закриття вікна, інтерфейс програми ховається в системний трей, але відтворення не переривається. Повністю вийти з програми можна кнопкою **Вихід** у головному меню або через контекстне меню трею.
* **Тема**:
  Швидке перемикання кольорової гами інтерфейсу. Доступна сучасна темна тема (для зменшення навантаження на очі ввечері) та класична світла тема.

### 3. Вкладка "Запис та Плейлисти"
* **Автоматичний запис при старті нового мовлення**:
  Якщо активна ця опція, програма автоматично починатиме запис щоразу, коли ви вмикаєте відтворення станції.
* **Кнопка Запису (Почати / Зупинити запис)**:
  Дублює керування записом з головного вікна. Запис починається у папку `records/YYYY.MM.DD/` з назвою файлу `hh.mm.ss_назва_радіостанції.mp3` (якщо прав недостатньо, зберігається у папку `Музика\UkrRadioOnline_records`).
  Кнопка запису при цьому змінює колір на активний червоний кружок `●` (а при зупинці повертається до початкового стану).
* **Розумний авто-розподіл записів**:
  Якщо під час запису змінюється назва пісні чи програми (метадані), програма автоматично створює новий файл і записує назву треку безпосередньо в ім'я файлу (наприклад, `15.30.00_KISS FM - Океан Ельзи - Той день.mp3`). Усі недопустимі для файлової системи символи автоматично вирізаються.
* **Відкрити папку з записами**:
  Швидкий доступ до папки, куди зберігаються ваші аудіозаписи.
* **Завантажити плейлист (.m3u)**:
  Завантажуйте власні плейлисти у форматах `.m3u` або `.m3u8` прямо з налаштувань. Всі успішно додані радіостанції зберігаються у налаштуваннях програми та будуть доступні в списку джерел під відповідними назвами навіть після перезапуску програми. Вбудовано захист від великих чи некоректних файлів.

### 4. Вкладка "Сповіщення"
Тут можна тонко налаштувати системні повідомлення програми. Ви можете вмикати або вимикати сповіщення для таких подій:
- Згортання / фоновий режим.
- Оновлення плейлистів.
- Статус відтворення.
- Обрив зв'язку.
- Статус запису.
- Відкривати папку з записами після зупинки.
- Повідомлення про автозапис при старті.

* **Збереження розмірів вікон**:
  Програма автоматично запам'ятовує розміри головного вікна, а також усіх діалогових вікон (Довідка, Налаштування тощо) та відновлює їх при наступному запуску.

## Інші інтегровані можливості

* **Швидкий вибір станцій та Улюблене**: Усі популярні українські хвилі (Українське Радіо, Радіо Промінь, Радіо Культура, Радіо ROKS, Радіо Байрактар, Радіо Казка, Радіо Класик, Радіо Релакс, Радіо Шансон, Хіт FM) вбудовані безпосередньо в основний список на початку меню вибору. Ви також можете натиснути на кнопку з **Зірочкою** (☆) біля назви будь-кої іншої станції, щоб додати її до списку **Улюблених** — такі станції відображатимуться одразу під основним списком.
* **Читання назв пісень (ICY метадані)**: Програма має власний парсер метаданих, що працює в режимі безперервного фонового з'єднання. Назви пісень та виконавців з інтернет-потоків оновлюються **миттєво в реальному часі** зі зміною треку на сервері. Це дозволяє здійснювати ідеальний авто-розподіл файлів запису без мікро-обрізок та втрат інформації. Дані про поточний трек відображаються статичним текстом під вибором джерела.
* **Динамічне оновлення плейлистів**: Програма автоматично у фоновому режимі перевіряє та завантажує свіжі плейлисти. Першочергово завантажуються офіційні стріми з сервера `https://radio.ukr.radio/` (включаючи регіональні станції та якість потоків), а потім плейлисти IPTV від `iptv.org.ua` та збірки `Opencartbot`. Це автоматично розширює перелік доступних хвиль до сотень варіантів.
* **Захист від запуску копій (Single Instance)**: Програма блокує запуск кількох екземплярів одночасно. При спробі відкрити додаток вдруге, перше вікно автоматично розгорнеться на екрані, а в треї з'явиться відповідне сповіщення.

## Запуск та Встановлення

Ви можете запустити додаток двома способами:

### Спосіб 1: Готовий Standalone Executable (Рекомендовано)
1. Перейдіть до папки `dist`.
2. Запустіть файл `UkrRadioOnline.exe`.
   - Це повністю автономна версія додатку, яка не потребує встановленого Python на вашому комп'ютері.

### Спосіб 2: Запуск із вихідного коду за допомогою скриптів
1. Переконайтеся, що на вашому комп'ютері встановлено **Python (версія 3.10 або вище)**.
2. Запустіть файл `_START_.bat` у корені проекту.
   - Цей скрипт автоматично перевірить наявність Python, створить локальне віртуальне середовище `.venv`, встановить необхідну бібліотеку `PyQt6` та запустить додаток.

При наступних запусках додаток відкриватиметься миттєво.

## Компіляція у виконуваний файл (.exe)

Для самостійного створення автономного виконуваного файлу:
1. Запустіть скрипт `_BUILD_exe_.bat`.
2. Скрипт перевірить середовище, за необхідності автоматично завантажить пакувальник UPX для стиснення виконуваного файлу та скомпілює проект за допомогою PyInstaller. Готовий файл буде збережено у папці `dist/`.

## Технічні вимоги

- ОС: Windows 10/11 (використовуються вбудовані системні кодеки Windows Media Foundation).
- Інтернет-з'єднання для відтворення радіостанцій.
- Додаткові плеєри чи кодеки (як-от VLC або FFmpeg) не потрібні — додаток є повністю самодостатнім та оптимізованим!
